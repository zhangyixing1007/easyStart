预格式

这里是没有用预格式的情况：

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello World"); } }

使用预格式的情况:

public class HelloWorld {
 
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
