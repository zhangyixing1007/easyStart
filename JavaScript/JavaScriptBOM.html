JavaScript BOM



BOM即 浏览器对象模型(Browser Object Model)。 浏览器对象包括
  • Window(窗口)
  • Navigator(浏览器)
  • Screen (客户端屏幕)
  • History(访问历史)
  • Location(浏览器地址)
